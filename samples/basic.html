Basic JSINQ example

The source code for this example can be found in the file basic.js.

This example showcases the use of jsinq.Enumerable without the use of the JSINQ query compiler. Queries are defined by chaining together method calls on an instance of jsinq.Enumerable. In this concrete example, an HTML table is filled from a JSON structure using a simple query statement. The query itself generates the HTML nodes that make up the rows of the results table.
