JSINQ DOM example

The source code for this example can be found in the file dom.js.

This example showcases the use of jsinq.Query. It operates on a set of HTML-DOM nodes which are taken from the body of the input table you see below. It will then select table rows by certain criteria and copy the matching rows into the results table below the input table.
